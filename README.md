# Smart Job Portal Backend

This is a backend project for a Smart Job Portal developed using Java Spring Boot and MySQL.

## 🚀 Features
- User Registration API
- Fetch All Users API
- RESTful API Development
- Database Integration using MySQL
- Spring Data JPA for ORM
- Basic Authentication using Spring Security

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL
- Maven
- Postman

## 📌 How It Works
- Client sends API request (Postman/Frontend)
- Spring Boot processes request via Controllers
- Data is stored/retrieved from MySQL database
- Response is returned in JSON format

## 🔗 API Endpoints
- POST /signup → Register User
- GET /api/users/all → Get All Users
